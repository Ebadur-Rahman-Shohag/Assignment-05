Question-1: What are the different ways to select an element in the DOM?

There are many ways to select an element in the DOM. Here are some example 1.getElementById() 2. getElementsByClassName() 3. getElementsByTagName 4. querySelector() 5. querySelectorAll() etc.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Here are some key differences between innerHTML, innerText, and textContent: 1. innerHTML gives us the HTML content of an element. We can also set the HTML content of an element by using innerHTML 2. innerText gives us the text content of an element. We can also set the text content of an element by using innerText 3. textContent and innerText are similar but textContent also gives the hidden text.

Question-3: What is event delegation in the DOM?

Instead of using eventListener() to every individual element we can use a single eventListener() to the parent container. This technique is called event delegation.

Question-4: What is event bubbling in the DOM?

When we use eventListener() in an element this evenetListener() is not only trigger on this element but also trigger on it's parent element and continues to move up until the document root is reached .

Question-5: How do you create, add, and remove elements using JavaScript?

1. We can create an element by using document.createElement() 2. We can add an element by using appendChild() method 3. We can remove an element by using removeChild() method
